FitLibraryWeb For FitNesse

This contains the first release of FitLibraryWeb for FitNesse users. It is based on Java 6.

Installing FitLibraryWeb

Before FitLibraryWeb can be used, you need FitNesse and FitLibrary to be installed:

  • If you need to, install FitNesse20090818 by following the instructions on the FitNesse web site.
  • However, if you've already been using that version of FitNesse, ensure you have a backup copy before you proceed.
  • If you haven't already done so, install FitLibrary20091020 on to the new image of FitNesse by following the instructions with that release.
Now you can install FiLibraryWeb
  • Drop the contents of folder fitnesse inside this FitLibraryWeb release onto the folder of your existing FitNesse system. That will copy the folders and files into the corresponding places in the FitNese release, so they are available throught FitNesse.
  • A Confirm Folder Replace dialog will appear: "This folder already contains a folders named 'FitNesseRoot, etc'. If the files in the existing folder have the same name as files in the folder you are moving or copying, they will be replaced. Do you still want to move or copy the folder?"
  • If you're happy to add the FitLibraryWeb files concerned, click Yes to All.
  • If not, click No.

Links on Front Page

To make it easy to access the FitLibraryWeb documentation and specifications, you may like to add the following line to the front page:
    |FitLibraryWeb|
